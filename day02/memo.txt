# 이미지 삽입 태그
<img src="파일경로" alt= "설명"/>
- src 속성은 넣고 싶은 이미지의 경로를 입력하며,  웹 상의 주소를 입력해도 된다
- alt 이미지에 대한 설명을 작성하며, 어떠한 이유로
이미지가 뜨지 않으면 설명이 대체되어 나온다.

# 링크를 거는 a태그(앵커 태그)
<a href="링크주소나 경로">링크를 걸어줄 요소</a>

# 인라인/블록 요소
<span></span> : 대표적인 인라인 태그
<div></div> : 대표적인 블록 태그
- 둘 다 의미가 없다.(시멘틱하지 않은 태그)

# 인라인 요소와 블록 요소
요소를 여러 관전으로 분류할 수 있다.
그 중 가장 기본이 되고 중요하게 봐야할 것은 인라인과 블록이다.

- 인라인 요소
* 내용물 크기만큼만 영역을 차지하므로 다른 요소와 같은 줄에 
  위치할 수 있다.
* 대표적 : span, strong, b 등이 있다.

- 블록 요소
* 내용물과 상관없이 한 줄을 다 차지하므로 다른 요소와 같은 줄에 
  위치할 수 없다.
* 대표적 : h1, div, p 등이 있다.
*블록 요스는 너비 속성이 100%를 다 차지하므로 자동으로 줄 바꿈 된다.

# CSS(Cascading Style Sheets)
HTML 로 만든 웹 문서의 스타일(디자인)을 담당한다.
- 스타일 이란 글꼴, 색상, 테두리, 정렬, 배치 방법등의 외관을 말한다.
- HTML에서 제공하는 속성으로 간단한 스타일 지정을 할 수 있다.
- 그러나 내용과 디자인이 분리되어 있어야 유지보수에 유리하므로,
  CSS를 별도로 사용하는것을 권장한다.

# CSS 작성 형식
속성1 : 속성값1;
속성2 : 속성값2;
속성은 미리 정해져 있으며 필요에 따라 검색하여 사용한다.

# 스타일을 적용하는 방법
1. 인라인 스타일
    스타일을 적용시킬 태그에 style 속석을 사용한다.
    <p style="속성1:속성값1; 속성2:속성값2;">내용</p>

    인라인 스타일은 간단하게 사용하기에는 좋으나 코드가 길어질 수록
    가독성이 떨어진다. css와 html이 뒤섞여 유지보수가 매우 힘들어진다.

2. 내부 스타일
    <head> 영역에 <style></style> 태그를 사용하여 각 요소들의 스타일을 정의한다.
    형식은 다음과 같다.

    선택자 {
        속성1 : 속성값1;
        속성2 : 속성값2;
    }

    선택자 : 어떤 요소를 선택하기 위해 사용하는 것


3. 외부 스타일
별도의 css파일을 만들어 적용시킨다.
css와 html을 완전히 분리하여 관리할 수 있기 때문에 가장 권장하는 방식이다.
css파일을 html파일에 적용시키기 위해서는 <head> 영역에 <link>태그를 사용한다
<link rel="stylesheet" href="css파일경로">


# id와 class
id와 class 속성은 특정 태그들을 식별하고 그룹화하는데 사용한다.
모든 태그에 사용 가능한 속성이다.

#id
특정 태그에 식별 가능한 고유한 값을 부여함
한 페이지 내에 같은 id가 있으면 안된다.
선택자로 사용할 때는 #id 로 사용한다.

#class
여러 태그를 하나의 집단으로 묶어줌
선택자로 사용할 떄는 .class 로 사용한다.

하나의 태그에 id와 class를 동시에 가질 수 있으며 class를 여러개 가질 수도 있다.


# Box Model
HTML의 모든 요소는 일정한 영역을 차지하게 된다.
이런 영역을 Box Model 이라고한다.

# 박스 모델의 구성
1. margin : 바깥 여백(테두리 밖의 여백)
2. border : 테두리
3. padding : 안쪽 여백 (테두리 안쪽 여백)
4. contents : 컨텐츠의 영역

인라인 요소의 박스 모델도 margin, border, padding 은 설정이 가능하지만
width, height 를 사용할 수 없으므로 contents 영역 크기를 설정할 수 없다.


# display 속성
HTML에서 요소를 어떻게 표시할지 결정한다.
HTML 요소들은 기본적으로 display의 속성값이 설정되어 있다.

블록 요소 -> display : block
인라인 요소 -> display : inline

특정 요소의 display속성을 수정한다면 블록 요소를 인라은으로,
인라인 요소를 블록으로도 수정이 가능하다.

# display 속성의 기본 4가지 값
- none
    요소를 보이지 않게 설정한다.
    단순히 투명해지는 것이 아니라
    아예 사라지게 되므로 공간을 차지하지 않는다.
- block
    블록 요소로 설정한다.
    블록 요소는 기본적으로 가로 영역을 모두 차지한다.
    너비와 높이를 조절할 수 있다.
- inline
    인라인 요소로 설정한다.
    인라인 요소는 기본적으로 컨텐츠의 크기만큼만 영역을 차지한다.
    블록 요소와 달리 컨텐츠의 높이, 너비를 설정할 수 없다.
- inline-block
    인라인-블록 요소로 설정한다.
    인라인의 특징과 블록의 특징을 섞어 놓은 요소이다.
    인라인 처럼 컨텐츠 만큼만 영역을 차지하므로 가로 배치가 가능하다.
    블록 처럼 컨텐츠의 너비, 높이를 수정할 수 있다.
    내부는 block요소 , 외부는 inline 요소의 특징을 갖는다.
